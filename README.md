# Mapled

![Mapled](./docs/assets/mapled-header.png)

One day I got bored and I ordered a LED matrix and a little board. Then I started experimenting, and eventually created Mapled.

Mapled, portmanteau of Maple 🍁 and LED 💡, designates the whole thingy: hardware and pieces of software.

Note : this repo is more a showcase and a way for me to organize my thoughts about this project rather than distribute ready-for-use code. Still, I hope it can be a source of inspiration for similar DIY projects.


## Features

- Displays time
- WiFi connectivity
- MQTT communications
- Remote control
- Home Assistant compatibility

## Hardware

- **Board :** Pimoroni Interstate 75 W (RP2350). Bought on PiShop.ca (32 CAD).
- **Display :** 64x32 P3 Hub75 LED matrix. Bought on AliExpress (15 CAD).
- **Power :** USB-C cable & power brick. Certainly bought on Amazon.
- **Others :** Mini easel for cute look. Bought at Dollarama (~2 CAD).

## Software

The I75W runs on [MicroPython](https://micropython.org/)-based firmware and comes with a handful of ready-to-use modules.
